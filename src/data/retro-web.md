# 回归2000年代的互联网

一直对本世纪初的互联网怀有一种特殊的向往。那时的网页没有花哨的渐变，没有圆角，也没有庞大复杂的单页动画。有的只是浅蓝色的导航栏，黑色的宋体或Tahoma文字，带有像素边缘的老式原生按钮。

## 为什么选择这种设计？

现代网页尽管美观，却总让人感到信息的**过载**与**浮躁**。

这套博客界面刻意采用了：
*   **深蓝 `#3b5998`** 这个互联网世界最具代表性的色彩之一。
*   简单的原生链接下划线特效。
*   极其方正的容器。
*   轻量且纯静态的内容展现方式。

希望在这里记录的想法，也能像这套UI一样，安静、克制、且长久。

> “在互联网的喧嚣中寻找一片宁静之地。”

这正是我制作这个轻量日记站点的初衷。
