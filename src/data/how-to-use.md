# 如何使用这个轻量级日志系统

欢迎来到你的全新个人中心空间。

这个系统没有数据库，不需要登录，所有的内容都在你本地的存储和代码库之中，非常适合希望掌握自己绝对数据主权的内容创作者。

### Markdown 支持的语法

你在日常编写日记的时候，可以充分利用 Markdown 的特性，例如：

#### 无序列表
* 记录灵光一现的想法
* 列出周度计划
* 书写读书笔记

#### 代码块记录
如果你是个程序员，可以很方便地记录代码片段：

```javascript
function sayHello() {
  console.log("Hello, Retro World!");
}
```

希望你在这里创作愉快！
